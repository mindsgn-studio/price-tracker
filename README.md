# Price Tracker

## Getting Started

```
    Note: this assumes you have go installed already. if not please install go: https://www.digitalocean.com/community/tutorial-collections/how-to-install-go
```

### 1. Fork repository and clone

### 2. Download mod

```
    go download
```

### 3. test

```
    go test
```

### 3. build

```
    go build -o scraper
```
